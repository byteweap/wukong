# Log Contrib

本目录提供 `component/log` 的实现。

## 实现列表

- zap: `github.com/byteweap/wukong/contrib/log/zap`
- logrus: `github.com/byteweap/wukong/contrib/log/logrus`
- zerolog: `github.com/byteweap/wukong/contrib/log/zerolog`
- fluent: `github.com/byteweap/wukong/contrib/log/fluent`
- aliyun: `github.com/byteweap/wukong/contrib/log/aliyun`
- tencent: `github.com/byteweap/wukong/contrib/log/tencent`

## 安装

```bash
go get github.com/byteweap/wukong/contrib/log/zap
```

替换为你需要的具体实现路径即可。
